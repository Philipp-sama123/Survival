# Survival

This will become a Unity Survival Game

## Installation

For setting up you will need: 
- Unity 2020.3.26f
- Cinemachine
- TextMeshPro
- **new** Unity Input System
- Amplify Animations
- Polyart Camp Site
- Survivor_Girl_1

## Usage

```python
pull Project 

import the necessary Assets

run the project!
```
